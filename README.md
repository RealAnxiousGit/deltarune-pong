# DELTARUNE Pong
A Deltarune fangame based on an arcade classic. Created using Google Gemini and Cursor.

# Controls
W/S - 1 Player
Up/Down - 2 Player
M - Mute 
Esc - Pause

# Features
- Custom music support!
- Fully made within HTML, CSS, and JavaScript
- Play with 2 players!
- Full on mobile support (Browser only)
- And much more!
